(static) STATE :Object

All the states StreamStash can be in - CONFIGURING: Instantiated and ready for input/output plugins and filters to be added - STARTED: Configured and ready to process events - STOPPING_INPUT: Beginning to shutdown, all input plugins should not be emitting events anymore - STOPPING_ALL: All inputs have stopped emitting events, waiting for in flight events and plugins to complete - STOPPED: All events and plugins have completed, we are stopped
